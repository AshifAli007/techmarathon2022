Bumble Jumble

Programming Language - C++
The event will consist of two rounds.

Event Description

First round

Time limit: 30 min
The first round will consist of 40-50 questions comprising snippets of code in the jumbled form, Small functions with fill-ups, MCQ, etc. Participants are required to answer the questions in specified formats. The top 3 teams solving the maximum number of correct questions in the stipulated time will qualify for the next round.

Second round

Time limit: 60 min
The second round will consist of two questions as follows
Question 1: Code having error/ Incorrect code (Time – 15 min)
Question 2: Code with jumbled or missing lines/ having errors. ( Time – 20 min )

Prize Distribution

The teams will be judged on the basis of accuracy and the amount of time taken.
