Code Benders

Code Benders is a coding contest that will be hosted on the CodeChef. There will be around 5 - 6 coding problems based on Problem Solving Skills, Data Structures and Algorithms.

Event Description

The Contest will only have a 1 stage format and no prelims and will be completely online. The contest will follow IOI Style Rank list which won’t result in any ties between the participants. The duration of the contest will be 3 hours and timings, as per CodeChef guidelines, need to be from 8:00 PM to 11:00 PM.

Prize Distribution

The top 2 participants will be awarded with prizes:
  1. Winner - ₹2500
  2. 1st Runner Up - ₹1500
